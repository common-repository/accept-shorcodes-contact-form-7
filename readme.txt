Contact Form 7 Addon

== Description ==

A simple addon to Contact Form 7 to enable external shortcodes within form and message body. 

== Installation ==

1. Upload the entire `contact-form-7-accept-shortcodes` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

Now you can use external shortcodes within your form and message body of Contact Form 7 Plugin.
